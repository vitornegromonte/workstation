## Workstation configure

Script para instalação automatizada no Ubuntu

Passo a passo:

	1 - cd Documents

	2 - git clone https://github.com/vitornegromonte/workstation.git

	3 - cd workstation

	4 - chmod +x vitor.sh

	5 - sudo ./vitor.sh
